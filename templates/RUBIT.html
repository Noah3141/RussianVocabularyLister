Russian Vocabulary List Maker

The most important part of learning language is not that you use the best methods for learning. It is that you use methods that can be sustained. Varying what tools you use goes a long way towards longevity in learning. While vocabulary lists in and of themselves are not incredible tools for language input, they can be very helpful for augmenting the usefulness of a text. Beginners and more advanced learners both can benefit from easily being able to quickly create vocabulary lists for unstructured, uncurated, naturally produced content.

This tool can be used to create vocabulary lists from YouTube transcripts of podcasts or speeches, book chapters, articles â€” any Russian text.

Input your study material text and choose settings for what kind of list output you want. You can set the "breadth" to include the "Full List" (all words), "Broad List" (words that occur more than once) and "Top Words" (words that occur [right now based on more than 4 times but it should be based on list size]). "Raw Vocabulary" will return all words, while "Verb Pairs" will return the aspectual pairs in your text, and "Verb Trees" will return verbs in the tree model, in the simplest form: Root Forms (-Branch Forms)
