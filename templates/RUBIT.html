List Maker

This tool can be used to create vocabulary lists from Russian text. Try YouTube transcripts of podcasts or speeches, book chapters, or articles.

Input your study material text and choose settings for what kind of list output you want. You can set the "breadth" to include:

  • "Full List" (all words)
  • "Broad List" (exclude the least common words)
  • "Top Words" (only the most common words)
  • "Bottom Words" (only the least common words)

"Raw Vocabulary" will return all words, while "Verb Pairs" will return the aspectual pairs in your text, and "Verb Trees" will return verbs in the tree model, in the simplest form: Root Forms (-Branch Forms)

All of the words will have links to Wiktionary/Викисловарь, which you can switch between using the button. If the key doesn't recognize a word, it will attempt to automatically update that word.




The Complete Verb Pair List page contains as close to a full representation of all core Russian verb pairs as possible.

The Complete Verb Tree List page contains these verbs parsed into the tree model described in The Tree Model. The trees are sorted so that the most proliferated trees are at the top, which are the highest yield vocabulary. These are statistically the best words to learn first.

You can utilize the example sentence generator linked in the footer to see vocabulary in context, separate from your study material. This lets you study the vocabulary quickly and effectively, so that when you turn back to the podcast, article, or book chapter, you can follow the material the first time through!
